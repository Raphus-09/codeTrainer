# 지판 트레이너 (Fretboard Trainer)

기타·베이스 지판과 키보드(건반)를 퀴즈로 익히는 웹앱. 설치나 빌드 없이 `index.html` 하나로 동작합니다.

**지원 악기: 기타(6현) · 베이스(4현) · 키보드(2옥타브 건반)** — 상단 토글로 전환하며, 퀴즈·커리큘럼·일일 학습·진행 기록이 악기별로 동작합니다. (코드 퀴즈·코드 사전은 기타 전용, 베이스 패턴은 베이스 전용, 지판 탐색은 기타/베이스, 건반 탐색은 공용)

## 실행 방법

- **가장 간단하게:** `index.html` 파일을 브라우저로 열기 (더블클릭)
- **로컬 서버로 (권장):**
  ```bash
  cd fretboard-trainer
  python3 -m http.server 8765
  # 브라우저에서 http://localhost:8765 접속
  ```

## 앱 구성 — 3개 파트

### 🎯 퀴즈 파트

| 모드 | 내용 |
|---|---|
| 🗺️ 커리큘럼 | 듀오링고식 스텝 학습 — 유닛 5개, 기타 18스텝/베이스 16스텝. 모든 스텝 자유 선택, 통과하면 ⭐ 완료 표시 (통과 기준·최고 기록 저장) |
| 📅 일일 학습 | 매일 12문제 (신규 유형 포함 배합), 완료 시 연속 학습 일수 적립 |
| 음이름 퀴즈 | 지판 → 음이름 |
| 음 위치 찾기 | 음이름 → 지판 (모든 위치 클릭) |
| 음정 퀴즈 | 두 지점 사이의 음정(인터벌) 맞히기 |
| 코드 퀴즈 | 지판 폼 → 코드 이름 (오픈 + 바레, 기타 전용) |
| 화음 퀴즈 | 지판에 쌓인 코드 톤 → 화음 이름 (maj/m/dim/maj7/7/m7) |
| 코드→구성음 | 코드 이름 → 구성음 모두 고르기 |
| 구성음→코드 | 구성음 목록 → 코드 이름 맞히기 |
| 코드 톤 짚기 | 코드 이름 → 루트·3음·5음(·7음)을 지판에서 차례로 짚기 |

### 🎵 리듬 파트

| 모드 | 내용 |
|---|---|
| 메트로놈·드럼 | BPM 입력(30~300)·슬라이더·프리셋, 4/4 클릭(첫 박 강세) 또는 8비트 드럼(킥·스네어·하이햇), 박자 시각 표시 |
| 탭 템포 | 탭(또는 스페이스바)으로 BPM 측정 — 4분음표(4비트)·8분음표(8비트) 기준 값 동시 표시 |

### 📚 배우기 파트

| 모드 | 내용 |
|---|---|
| 지판 탐색 | 스케일·아르페지오·크로매틱을 루트/도수 색상으로 시각화, 클릭하면 소리 재생 |
| 코드 사전 | 루트·종류를 고르면 수록된 모든 폼 + 구성음 표시, 소리 듣기 (기타 전용) |
| 베이스 패턴 | 이동 가능한 운지 모양 — 트라이어드 3종·세븐스 아르페지오 3종·스케일 박스 4종 (베이스 전용) |
| 키보드 | 2옥타브 피아노 건반에 스케일·코드 톤을 도수 색상으로 시각화, 클릭 재생 |
| 학습 가이드 | 학습 순서, 옥타브 셰이프, CAGED, 도수로 보는 화성학, 리듬 연습법 |

## 특징

- 기타(6현)·베이스(4현) 표준 튜닝 지원
- **오답 가중 출제**: 틀린 문제일수록 더 자주 나옵니다 (기록은 브라우저 localStorage에 저장)
- **일일 학습 스트릭**: 매일 완료하면 연속 일수가 쌓입니다 (악기별 별도 기록)
- Web Audio로 정답 시 실제 음/코드/아르페지오 소리 재생
- 세션 점수·연속 정답 스트릭 표시
